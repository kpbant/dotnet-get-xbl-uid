# dotnet-get-xbl-uid (.NET Core 2.2) / Solution to retrieve UID for XBL.

### To Run:
```sh
authHeader = getXblAuthorizationHeader("email", "password");
if(authHeader != null) getXblUid("user", authHeader);
Console.ReadKey();
```
